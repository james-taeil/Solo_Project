# Solo_Project — 로또 번호 생성기

한국 로또 6/45 번호를 무료로 생성하는 웹 서비스입니다.

## 페이지

- `index.html` — 로또 번호 생성기 (메인)
- `guide.html` — 이용 가이드 / FAQ
- `about.html` — 사이트 소개
- `contact.html` — 문의하기
- `privacy.html` — 개인정보처리방침
- `styles.css` — 공통 스타일
- `script.js` — 번호 생성 로직
- `site.js` — 공통 UI
- `ads.txt` — Google AdSense 인증
- `sitemap.xml` / `robots.txt` — SEO

## 로컬 실행

```bash
python3 -m http.server 8765
```

브라우저에서 `http://localhost:8765` 접속
